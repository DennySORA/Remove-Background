# Remove Background

这是一款易用的图片去背工具，提供多种方案与引导式流程，适合批量处理，也能应付普通照片与绿幕照片。

## 使用方案
- 引导式流程：选择文件夹、选择方案、调整强度、确认后开始处理。
- 绿幕流程：适合纯绿背景，让边缘更干净并减少色彩残留。
- 普通照片流程：适用于人像、商品与混合背景的照片。
- 速度与质量取舍：可根据时间与成品质感选择模式。

## 功能介绍
- 多种去背方案与清晰的说明。
- 批量处理，并提供进度与结果摘要。
- 可调整强度，控制去背程度。
- 输出透明背景，便于后续使用。

## 外部依赖（第三方）
- **backgroundremover** — 外部依赖。背景移除方案之一。
- **rembg** — 外部依赖。另一种背景移除方案。
- **transparent-background** — 外部依赖。背景移除方案。
- **onnxruntime** — 外部依赖。支持上述方案的运行环境。
- **pillow** — 外部依赖。图像读写工具。
- **moviepy** — 外部依赖。列于依赖清单的媒体工具。
