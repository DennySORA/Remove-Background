# Remove Background

複数の方式とガイド付きの流れを備えた、使いやすい背景除去ツールです。まとめて処理したい場合や、一般写真・グリーンスクリーン写真の両方に対応します。

## 利用オプション
- ガイド付きの対話的な流れ：フォルダを選択し、方式を選び、強さを調整してから確認して処理します。
- グリーンスクリーン向け：単色の緑背景に最適化し、縁をきれいにし色かぶりを抑えます。
- 一般写真向け：人物、商品、混合背景の写真に対応します。
- 速度と品質のバランス：時間や仕上がりの期待に合わせて選べます。

## 主な機能
- 複数の背景除去方式をわかりやすく提示。
- 一括処理と進捗・結果サマリー。
- 強さ調整で除去の度合いをコントロール。
- 透明背景の出力で後工程に使いやすい。

## 外部依存（サードパーティ）
- **backgroundremover** — 外部依存。背景除去の選択肢の一つ。
- **rembg** — 外部依存。複数の選択肢を持つ背景除去方式。
- **transparent-background** — 外部依存。背景除去方式。
- **onnxruntime** — 外部依存。上記方式を支える実行環境。
- **pillow** — 外部依存。画像の読み書きユーティリティ。
- **moviepy** — 外部依存。依存関係に含まれるメディア用ライブラリ。
